# Signal Politics Institute — Public Archive

Public repository for the Signal Politics Institute (SPI).

This repository contains public-facing research outputs, dashboards, reports, publication assets, and approved methodology notes.

## Repository structure

- `dashboards/election/` — election dashboards
- `dashboards/conflicts/` — international conflict dashboards
- `dashboards/signals/` — core signals and issues dashboards
- `reports/` — Korean and English public reports
- `images/` — logos, thumbnails, and shared media assets
- `docs/methodology/` — approved public methodology documents
- `docs/public-notes/` — public notices and explanatory notes

## Publication boundary

The default principle is public disclosure. Only core proprietary assets remain private, including source code, credentials, internal automation, unreleased model logic, raw internal data, risk controls, and investment-system implementation details.
